About

rlightbox is a jQuery UI mediabox that can display many types of content such as images, YouTube videos and Vimeo content. It has many features and is highly customizable.

Getting Started

To begin with, you have to add some JavaScript code that are required by the widget. Since rlightbox is a jQuery UI widget, and because it utilizes only minimal amount of dependencies, only few scripts are required. Remember that jQuery UI library needs basic jQuery library too.

In below examples, replace path with the path to the folder where rlightbox is kept: 

<script type="text/javascript" src="path/widget/lib/jquery-1.6.2.min.js"></script>
<script type="text/javascript" src="path/widget/lib/jquery.ui.core.min.js"></script>
<script type="text/javascript" src="path/widget/lib/jquery.ui.widget.min.js"></script>
<script type="text/javascript" src="path/widget/lib/jquery.ui.rlightbox.min.js"></script>

As you can see, version of jQuery library points to 1.6. rlightbox is built on version 1.6 and makes use of many features that previous versions of the library don’t have. 1.6 is the basis, but of course, you can use newer or updated versions when they are released.

You also have to add two style sheets to your page. Add these two lines in the head section: 

<link type="text/css" rel="stylesheet" href="path/widget/css/ui-lightness/jquery-ui-1.8.16.custom.css" />
<link type="text/css" rel="stylesheet" href="path/widget/css/lightbox.css" />

rlightbox shares the theme used by other jQuery UI widgets on your page. Therefore it is highly customizable. In above example I used ui-lightness theme, but you can use other: predefined, or even made by yourself! More on this in the Customization section.

But how to run the widget? Assume that you have 3 images and want to group it. A group of content is called a set in rlightbox terminology (you can find out more on sets in Sets section of this documentation). First you have to add anchors (a elements) with URLs to these images: 

<a href="flowers/tulip.jpg" title="a tulip">a tulip</a>
<a href="flowers/rose.jpg" title="a rose">a rose</a>
<a href="flowers/daisy.jpg"title="a daisy">a daisy</a>

As you can see, this is standard HTML code. To make a set (group them), add the following example class to them: lb_flowers: 

<a href="flowers/tulip.jpg" title="a tulip" class="lb_flowers">a tulip</a>
<a href="flowers/rose.jpg" title="a rose" class="lb_flowers">a rose</a>
<a href="flowers/daisy.jpg"title="a daisy" class="lb_flowers">a daisy</a>

Next we have to install rlightbox. To do this, add the following code: 

<script type="text/javascript">
    jQuery(function($) {
	$( ".lb_flowers" ).rlightbox();
    });
</script>

Now rlightbox is active on these anchors, all these images are grouped and clicking on one of them allows you to go to the others (with a mouse or a keyboard).

Optionally you can modify the default options to better suit you. As an example we will change the delimiter that separates counters. Instead of / we will use of: 

<script type="text/javascript">
    jQuery(function($) {
	$( ".lb_flowers" ).rlightbox({counterDelimiter: " of "});
    });
</script>

Now the 2nd image in the flower set is shown as 2 of 3.
To find out more about options, have a look at Configuration Options section.

Features

Panorama

Panorama is a way to display part of a full-sized image on the screen. Since often an image is larger than the screen, it has to be scaled down. Panorama allows you to to see it in its native resolutions by panning, i.e. drag & drop.

Panorama mode is available when the icon with four arrows pops up at the top left corner of the widget. To turn it on, you can either click on the icon or use keyboard ([Z] key is default).

Live Re-size

To see what Live Re-size is, open the following image and re-size the browser window a few times.

YouTube support

rlightbox supports playing YouTube videos. To add a YouTube video, use standard YouTube URL scheme like http://www.youtube.com/watch?v=PLtGMqvazzo. Short YouTube URLs are not supported. Additional parameters are not taken into account.

Vimeo support

rlightbox supports playing Vimeo videos. To add a video use either URL like this: http://vimeo.com/13763341 or this: http://vimeo.com/groups/polska/videos/3116167. Additional parameters are not taken into account.

Flash swf video

rlightbox also supports playing standard swf videos.

Nice error handling

When an error occurs, the user-friendly Error Screen appears to help you decide what to do!

Many widgets, one theme!

rlightbox is very customizable and its appearance shares a theme with other jQuery UI widgets.

Keyboard Navigation

Nearly no need to use your mouse! You can even customize keyboard keys to better suit you!

How to use

Opening

To open rlightbox just click on an anchor rlightbox was initiated on.

Closing

There are three ways to close the lightbox. You can either click the close icon (at the bottom bar), or click the overlay (it’s a darker background behind the widget), or use keyboard (default are [C] and [Esc] keys).

Opening next/previous content

In order to go to next or previous content you can either use your mouse (in case of images), or keyboard (images, videos). The default keyboard keys to open next content are [→] and [N] key. To go to previous content use [←] or [P] key.

Zooming aka Panorama

To zoom in/out an image (enable or disable Panorama mode) use either icon with four arrows (when it appears) or press [Z] key.

Sets

Set is a group of images, YouTube videos, Vimeo videos or a mix of them all. There may be many sets on a page. Elements from one group doesn’t have to be placed one after another (e.g in a div container) but can be placed in different places on a page. But it is important they should share the same name of a class that consists of a common prefix followed by a set name. In the below example we have 3 images grouped into one set called flowers. The set name is preceded by a prefix (lb in this example) followed by the underscore character. 

<a href="flowers/tulip.jpg" title="a tulip" class="lb_flowers">a tulip</a>
<a href="flowers/rose.jpg" title="a rose" class="lb_flowers">a rose</a>
<a href="flowers/daisy.jpg"title="a daisy" class="lb_flowers">a daisy</a>

The Error Screen

The Error Screen appears at times when there is an error. :) An error may appear when: a URL is broken, data can’t be fetched from a server for some time, and sometimes it is dependent on a browser. The default Error Screen looks like this. You can change the message and labels on the buttons.

Try again button is used to give loading another try. Sometimes when a server is busy and data can’t be fetched for 5 seconds, the error occurs. More than often Try again button does its trick.

Reject this content is useful when consecutive trial of loading content failed. It removes such content from a set and uninstalls rlightbox from the anchor.

Setting Size of Content

Images

There is no need to set size for an image. rlightbox uses its native size.

YouTube, Vimeo

Use videoHeight and videoWidth options to set size of this kind of content.

Flash swf video

If you do not want swf video to use videoHeight and videoWidth you can overwrite them adding parameters at the end of the URL: 

<a href="http://www.adobe.com/products/flashplayer/include/marquee/design.swf?width=790&height=290" title="flash video">Flash video</a>

Only integers are accepted – no width=42px please!

Configuration Options

Option Type Default value Description
againButtonLabel string "Try again" Label on the left button in the error screen.
animationSpeed string ["fast" | "slow" | "normal"] | integer "fast" How fast rlightbox is to appear, re-size, etc. Values are normal values used in jQuery. Integer should be given in milliseconds.
counterDelimiter string " / " Separates current content number in a set from the total number
errorMessage string "Oh dear! Something went wrong! If the problem still appears let the page’s admin know. Would you like to try again or reject the content?" The message which appears when there is an error while loading content or when a URL is broken. Sometimes an error may appear unexpectedly and it depends on a browser.
keys object {next: array, previous: array, close: array, panorama: array} {next: [78, 39], previous: [80, 37], close: [67, 27], panorama: [90, null]} Codes pointing to real keyboard keys. To determine which code is a key, use this page.
overwriteTitle boolean false Overwrites original title taken from Vimeo or Youtube video when title attribute is provided.
rejectButtonLabel string "Reject this content" Label on the right button in the error screen.
setPrefix string "lb" What prefix followed by ‘_’ and a name use to group (make a set) content. To group 2 images, add a new class, e.g. ‘lb_flowers’ to both of them.
showMap boolean true Indicate whether to show or not mini map in Panorama mode.
videoHeight integer 385 Height of a video. It may denote maximum value.
videoWidth integer 640 Width of a video. It may denote maximum value.

Compatibility

rlightbox was tested and has been tested in many browsers. If you come across a bug, please report it in the bug tracker. Some browsers like Internet Explorer 6 are too old and will not be supported. Because of some browsers release cycle like that of Chrome and Firefox until recently, it is very hard to keep track of rlightbox behaviour in these browsers. Thus the widget was tested just in Chrome/Chromium and Firefox 4 and 5. Also please note, it is not so easy to estimate the number of browsers to test in especially when it comes to minor releases or a bug fix releases. The more the better is not the answer. If the browser you want to use rlightbox in, is for some reason important to you, and the widget doesn't work in it, you can fill in a bug report in the bug tracker.

rlightbox was tested in the following browsers:

  • Internet Explorer 7+
  • Firefox 3.0.19+
  • Chrome
  • Opera 10.00+
  • Safari 4.0.5+

Customization

Lightness Blitzer Smoothness Start Sunny

rlightbox is very customizable. You can suit themes to your needs. As an exercise, choose any above theme and open any example available on the page. See? This is only a tiny example of what you can achieve!

How to use a new theme

First go to the jQuery UI theme gallery. To the left you can see ThemeRoller – a tool to help you choose themes from and to make your own. Then click on the Gallery tab. It features predefined themes. Clicking on thumbnails allows you to see live preview of a theme. Now click on the Download button. When new page opens, just use Download button again – I recommend you to select all components first. When it downloads, unzip the file, go to the css folder and copy the folder with the theme to the folder where rlightbox is kept: …/widget/css/. Now all you have to do is to add the link to the new theme to your page, for example: 

<link type="text/css" rel="stylesheet" href="path to the rlightbox/widget/css/sunny/jquery-ui-1.8.16.custom.css" />
